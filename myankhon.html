DAO Tax Collection System (Myan Khon Solution)

Description

Myanmar’s first tax digitizing, tax collection process beginning to end for municipal governments. It is changing the computerized system from their current paper-based system. In the system, mobile apps for tax collectors and web application for other executive levels.

Tasks and Responsibility

  • Workflow analysis about the whole municipal government’s tax collection and management process
  • Database structure and system architecture
  • Backend API for mobile app, Admin Portal development
  • Data dashboard and KPI dashboard for data analysis
  • Customizable reports
  • Using Quartz Scheduler library to control thousands of record operations (Real-Time)
